# 🚛 Парсер автошкол РФ — категория C и выше

Собирает данные обо всех автошколах России, где можно обучиться на грузовые автомобили:
категории **C**, **C1**, **CE**, **C1E**.

## Источники данных

| Источник | Охват | Данные |
|---|---|---|
| [avtoshkoli.ru](https://avtoshkoli.ru) | ~80+ городов РФ | Название, адрес, телефон, цена, рейтинг |
| [гибдд.рф/drivingschools](https://гибдд.рф/drivingschools) | Все регионы РФ | Официальный реестр с заключениями ГИБДД |

После сбора данные **дедуплицируются** по ключу `город + название`, категории — объединяются.

## Структура проекта

```
driving-school-parser/
├── main.py                    # точка входа
├── requirements.txt
├── src/
│   ├── parser_avtoshkoli.py   # парсер avtoshkoli.ru
│   ├── parser_gibdd.py        # парсер реестра ГИБДД
│   └── exporter.py            # дедупликация + экспорт CSV/JSON
└── output/                    # результаты (создаётся автоматически)
    ├── driving_schools_cat_c.csv
    └── driving_schools_cat_c.json
```

## Установка

```bash
git clone https://github.com/YOUR_USERNAME/driving-school-parser.git
cd driving-school-parser
pip install -r requirements.txt
```

## Запуск

```bash
# Полный прогон всех источников и городов
python main.py

# Только один источник
python main.py --source avtoshkoli

# Только несколько городов
python main.py --cities moskva kazan ekaterinburg

# Тестовый прогон (5 городов, без ГИБДД)
python main.py --dry-run
```

## Формат результата

### CSV (`driving_schools_cat_c.csv`)

| Поле | Описание |
|---|---|
| `name` | Название автошколы |
| `city` | Город |
| `address` | Адрес |
| `phone` | Телефон(ы) |
| `categories` | Категории через запятую (C, CE, C1, C1E) |
| `price` | Стоимость обучения |
| `rating` | Рейтинг |
| `reviews` | Количество отзывов |
| `source` | Источник данных |
| `source_url` | URL страницы |

### JSON (`driving_schools_cat_c.json`)

Аналогичная структура, массив объектов.

## Категории прав

- **C** — грузовые автомобили массой > 3 500 кг
- **C1** — грузовые 3 500–7 500 кг (открывается автоматически при C)
- **CE** — грузовик + прицеп > 750 кг (нужен стаж C ≥ 1 года)
- **C1E** — грузовик C1 + тяжёлый прицеп, суммарно до 12 000 кг

## Требования к возрасту и стажу

- **C / C1** — обучение с 17 лет, права с 18 лет
- **CE / C1E** — минимум 19 лет, стаж по C не менее 1 года
